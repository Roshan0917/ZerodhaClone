# 📈 Zerodha Clone - Full Stack Trading Platform

A complete **Full Stack Zerodha Clone Web Application** built to simulate a real-world stock trading platform.

This project provides a trading dashboard experience where users can create an account, login, manage funds, view holdings, track positions, manage orders, and analyze their portfolio.

The main goal of this project was to build a scalable full-stack application with modern frontend, backend, database, and deployment technologies.

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone <your-repository-link>
```

Move into project directory:

```bash
cd Zerodha-Clone
```


---

# Backend Setup

Go to backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create `.env` file:

```
MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

Run backend:

```bash
npm start
```

Backend will start running on your configured port.

---

# Frontend Setup

Go to frontend folder:

```bash
cd frontend
```

Install packages:

```bash
npm install
```

Run application:

```bash
npm run dev
```

---
